# LemonCraft

## How to setup

clone the repo

```
git clone https://github.com/lokeshkavisth/lemon-craft.git
```

cd into the cloned project

```
cd lemon-craft
```

install all the packages

```
npm install
```

run the server

```
npm run dev
```

Go to your browser and open `http://localhost:5173` you should see your project is running there.
